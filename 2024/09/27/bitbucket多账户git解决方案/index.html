生成多个公私密钥对

公钥填入对应账户

https://bitbucket.org/account/settings/ssh-keys/

add key

修改host

C:\Windows\System32\drivers\etc\hosts追加

域名和私钥映射

C:\Users\<你的用户名>\.ssh追加

使用

个人的账户还用 git clone git@bitbucket.org:xxx/xxx.git

公司账户就用 git clone git@company:xxx/xxx.git
